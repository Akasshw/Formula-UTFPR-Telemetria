# Formula-UTFPR-Telemetria
Dados utilizados em testes na equipe de fórmula student da universidade UTFPR.

Código de telemetria em MATLAB utilizado junto dos arquivos dos testes específicos de suspensão/direção.

Obs: Quando utilizar o código ele irá dar um erro no terminal, esse erro é devido a falta de um Add-on, suba a mensagem de erro do terminal e baixe o add-on que está sublinhado em azul.
